# Temas Puntuales

En esta sección aprenderemos a dar nuestros primeros pasos en Nuxt, con diferentes técnicas de desarrollo, renderizado y explicaciones del file system.

## Puntualmente veremos:

- Estilos globales
- Estructura de directorios
- Single Page Application - SPA
- Sitios estáticos
- SEO Metatags
- Páginas de error personalizadas
- Server side rendering - SSR
- NuxtLinks
- Despliegues como SPA y estático
- Rutas padre e hijas
- Layouts
- Configuraciones globales de Nuxt

Es una sección que nos enseñará muchas cosas de Nuxt.
