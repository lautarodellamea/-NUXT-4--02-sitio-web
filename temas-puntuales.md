# Temas Puntuales

## 1. Primeros pasos en Nuxt

En esta sección aprenderemos a dar nuestros primeros pasos en Nuxt, con diferentes técnicas de desarrollo, renderizado y explicaciones del file system.

### Puntualmente veremos:

- Estilos globales
- Estructura de directorios
- Single Page Application - SPA
- Sitios estáticos
- SEO Metatags
- Páginas de error personalizadas
- Server side rendering - SSR
- NuxtLinks
- Despliegues como SPA y estático
- Rutas padre e hijas
- Layouts
- Configuraciones globales de Nuxt

Es una sección que nos enseñará muchas cosas de Nuxt.

## 2. Integración de NuxtUI

En esta sección aprenderás a integrar NuxtUI, la librería oficial de componentes para Nuxt, con el objetivo de acelerar la construcción de interfaces modernas, consistentes y altamente personalizables. Vamos a trabajar desde la instalación hasta el diseño de pantallas completas tanto del sitio público como del panel administrativo.

### Instalación, configuración y componentes base

- Cómo instalar y configurar NuxtUI correctamente en tu proyecto.
- Uso de layouts y manejo de errores dentro de NuxtUI.
- Implementación de componentes base como botones, formularios y tarjetas.
- Personalización del tema global y variables de estilo.
- Cómo utilizar Iconify como sistema de íconos integrado.

### Construcción de interfaces y buenas prácticas

- Construcción del footer, pantallas de inicio y páginas de autenticación.
- Diseño de la interfaz del panel administrativo usando componentes de NuxtUI.
- Manejo de atributos especiales y uso de props y non-props.
- Creación de secciones dinámicas como marquee, separadores y elementos decorativos.
- Buenas prácticas para mantener la UI limpia, clara y escalable.
